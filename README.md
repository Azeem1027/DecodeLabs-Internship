#Project 2  Data Classification  Using AI
## About the Project
An AI-based data classification project implemented in Google Colab that uses Logistic Regression to classify plant species. Utilizing Python and the classic Iris dataset from `scikit-learn`, the model splits the dataset for training and testing, fits a multi-class Logistic Regression model, and evaluates performance using accuracy scores and detailed classification reports.
### Technologies Used
* **Python** (Google Colab Environment)
* **Pandas** – For data structuring and profiling
* **Scikit-learn** – For dataset loading, data splitting (`train_test_split`), model training (`LogisticRegression`), and evaluation metrics
